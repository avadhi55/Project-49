CSS stands for Cascading Style Sheets.

  • CSS describes how HTML elements should be displayed.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.

JS stands for Java Script.

  • JS is a programming language that allows you to implement complex things on web pages
  • JS displaying timely content updates, interactive maps, animated 2D/3D graphics, scrolling video jukeboxes.

HTML stands for Hyper Text Markup Language.

  • HTML is the code that is used to structure a web page and its content.
  • HTML is a markup language that defines the structure of your content.
